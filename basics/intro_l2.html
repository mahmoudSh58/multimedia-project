Introduction :

lesson 2: Variables and Data types

Variables are containers for storing data values.

In C++, there are different types of variables (defined with different keywords), for example:

  • int : stores integers represent in 4 byte ,from 232 to -232.
  • long long : stores integers like int but large represent in 8 byte ,from 264 to -264.
  • float : stores floating point numbers like (1.999) ,represent in 8 byte.
  • double : stores floating point numbers like (1.9999999) ,represent in 16 byte.
  • char : stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes represent in 1 byte.
  • string : stores text, such as "Hello World". String values are surrounded by double quotes ,is dynamic array of char type.
  • bool : stores values with two states: true or false.

Declaring Variables :

type variableName = initial_value;

 

  int myNum = 15;

 

  string str; //note no initial_value

 

  bool b = 1; //note non-zero value equal true and zero equal false

 

  float f1,f2,f3; // note can declare multivariable by ‘,’

 

 

Assignment Variables :

assignment variables by '=' operator

 

   int Num1;

 

   double Num2;

 

   Num1 = 15;

 

   Num1 = Num2; //note can assign int to double

 

 

Display Variables :

The cout object is used together with the << operator to display variables.

 

   int myAge = 35;

 

   cout << "I am " << myAge << " years old.";

 

   // To combine both text and a variable, separate them with the << operator

 

 


  Output :

 

  I am 35 years old.
