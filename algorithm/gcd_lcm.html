GCD and LCM

GCD

GCD between two or more numbers is the Greatest Common Divisor that divides these numbers without reminder.

Ex :

gcd(9, 15) = 3
gcd(10,20)=10
gcd(12,18)=6
gcd(12,8,16)=4

Two way to calculate GCD :

First way : prime factorization

This is done by taking the prime factors with the min power

Ex :

48 = 24 * 3
180 = 24 * 32 * 5
gcd(48, 180) = 2min(4,2) * 3min(1,2) * 5min(0,1) =22 * 31 * 50 = 12

 

   #include <bits/stdc++.h>

 

   using namespace std;

 

   vector <int> sev(int n){

 

       vector <bool> b(n+1,1);

       vector <int> p;

       b[0]=b[1]=0;

       for(int i=0;i*i<= n;i++){

           if(b[i]){

               for(int j=i*2;j<=n;j+=i) b[j]=0;

           }

       }

       for(int i=0;i<=n;i++) if(b[i]) p.push_back(i);

       return p;

 

   }

 

   vector <int> fact(int n , vector <int> p){

 

       vector <int> f;

       int i=0;

       while(n!=1){

           if(n%p[i]==0){

               n/=p[i];

               f.push_back(p[i]);

           }

           else i++;

       }

       return f;

 

   }

 

   int gcd(int a ,int b){

 

       vector <int> p = sev(max(a,b));

       vector <int> a_f = fact(a,p);

       vector <int> b_f = fact(b,p);

       cout << "a = ";

       for(int x : a_f) cout << x << ' ';

       cout << '\n';

       cout << "b = ";

       for(int x : b_f) cout << x << ' ';

       cout << '\n';

       int g=1 , n=a_f.size() , m=b_f.size() ,i=0,j=0;

       while(i< n && j < m){

           if(a_f[i]==b_f[j]){g*=b_f[j];i++;j++;}

           else if(a_f[i]>b_f[j])j++;

           else i++;

       }

       return g;

 

   }

Second way : Euclidean Algorithm

A more efficient way to calculate the GCD . This is done recursively by replacing (a, b) by (b, a mod b) repeatedly until the pair is (d, 0), where d is the greatest common divisor. The time complexity for this method is log(min(a,b))

Ex :

gcd(48,18) → gcd(18 , 48%18) = gcd(18, 12)
gcd(18,12) → gcd(12 , 18%12) = gcd(12, 6)
gcd(12,6) → gcd(6 , 12%6) = gcd(6, 0)

so : gcd(48,18) = 6

 

  ll gcd(ll a, ll b){

             ll(b==0) return a;

             return gcd(b, a%b);

   }

LCM

LCM two or more numbers is the Least Common Multiple that can be divided by these numbers without reminder.

Ex :

lcm(4, 6) = 12
lcm(10,20) = 20
lcm(1,5) = 5
lcm(2,3,4)=12

Two way to calculate LCM :

First way : iterate over the multiples

The naive solution for calculating the LCM between two integers is iterate over the multiples, and take the first common multiples. This take O(a*b) time complexity.

lcm(4,6) → multiples of 4 : 4 8 12 16 20 24
multiples of 6 : 6 12 24

 

Code :

 

   ll lcm(ll a ,ll b){

       for(ll i=max(a,b);i<= a*b ;i++){

           if(i%a==0 && i%b==0) return i;

       }

   }

 

 

Second way : use GCD

Another Faster solution is use the following rule:
lcm(a,b) = a*b / gcd(a,b)

This means that we remove the factors that we don't need, the factors that are common between them, which is the gcd
4's factors : 2 2
6's factors : 2 3
They have 2 as a common factor which is the gcd between 4,6

 

Code :

 

   ll lcm(ll a, ll b){

          return a/__gcd(a,b)*b;

   }
